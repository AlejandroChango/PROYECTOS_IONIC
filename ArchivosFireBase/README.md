# Cargar archivos en storage en Firebase

## Instalacion 

Instala ionic

```bash
  npm i -g @ionic/cli 
```

Clonar el repositorio

```bash
https://github.com/AlejandroChango/PROYECTOS_IONIC.git
```

Ir a la carpeta del proyecto

Instala las dependencias

```bash
  npm install
```

Inicia el servidor

```bash
  ionic serve
```
