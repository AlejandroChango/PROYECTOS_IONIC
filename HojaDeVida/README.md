# Aplicación de hoja de vida (CV)

Aplicación de hoja de vida (CV)
 
## Instalacion 

```bash
  npm i -g @ionic/cli 
```


Clonar el repositorio

```bash
https://github.com/AlejandroChango/PROYECTOS_IONIC.git
```

```bash
  npm install
```

Inicia el servidor

```bash
  ionic serve
```
