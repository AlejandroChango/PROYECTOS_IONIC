# Galeria APP 

## Instalacion 

```bash
  npm i -g @ionic/cli 
```


Clonar el repositorio

```bash
https://github.com/AlejandroChango/PROYECTOS_IONIC.git
```
colocarse en la carpeta Galeria-Storage

```bash
  npm install
```

Inicia el servidor

```bash
  ionic serve
```
